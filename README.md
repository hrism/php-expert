# php-expert

Modern PHP development skill for Claude Code.

## Features

- PHP 8+ language features (union types, enums, attributes, match expressions)
- Memory-efficient data processing with generators and SPL data structures
- Architecture patterns (DI Container, Repository, Service Layer)
- Database optimization (transactions, N+1 prevention, cursor pagination)
- Error handling with custom exception hierarchies
- Security best practices (validation, password handling)
- Performance profiling

## Installation

### npx skills

```bash
npx skills add hrism/php-expert
```

### Manual

Copy `SKILL.md` to `~/.claude/skills/php-expert/`

## Usage

The skill activates automatically when working on PHP projects.
